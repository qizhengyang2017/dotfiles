# Rime

## 快速上手

~/Library/Rime

- default.custom.yaml: 候选词个数、caps键功能
- squirrel.custom.yaml: 主题设置

default.custom.yaml中的设置会覆盖default.yaml。

修改之后按deploy重新部署，生效


## 现在使用的配置

https://github.com/ssnhd/rime?tab=readme-ov-file

- 该配置可以解决vim 中英文切换
- 会显示很多emoji

按下esc自动切换为英文模式。

| 按键                  | 功能                 |
| ------------------- | ------------------ |
| F4/ctrl+shift+grave | 方案选择（default.yaml） |
| =                   | 下一页                |
| -                   | 上一页                |
| tab                 | 分词                 |
| ;                   | 第二个候选              |
| "                   | 第三个候选              |
| 空格                  | 第一个                |
